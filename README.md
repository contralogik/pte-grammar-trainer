# PTE Grammar Trainer

An offline-first React + Vite + TypeScript app for PTE grammar practice.

## Features

- Dashboard with daily practice count, accuracy, weak grammar points, and 7-day activity.
- Grammar topic practice across 17 PTE-relevant grammar categories.
- Four question types: error correction, fill in the blanks, sentence combination, and reorder paragraph.
- Explanation page with Chinese explanation, English rule, common mistake, PTE relevance, and memory tip.
- Mistake book saved in `localStorage`, with review counts and topic filtering.
- Progress page with overall accuracy, topic accuracy, 7-day practice volume, and weakest topics.
- 1000 built-in mock questions with difficulty split: Easy 20%, Medium 50%, Hard 30%.

## Local Development

```bash
npm install
npm run dev
```
